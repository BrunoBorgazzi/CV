# 🎮 Bruno Borgazzi - Interactive Game Designer CV

Welcome to the repository of my interactive and responsive Curriculum Vitae. This project serves as a live, dynamic portfolio showcasing my profile, skills, and experience as a Mid-level Game Designer specialized in ARPGs and RPGs.

🔗 **[View the Live Portfolio Here](https://brunoborgazzi.github.io/CV/)**

## ✨ Features & Functionality

This landing page was designed to be much more than a static document. It includes several interactive features to provide a premium user experience:

- **🌍 Auto-Language Detection & Translation:** A custom-built Vanilla JavaScript translation engine automatically detects your browser's language. It serves the page in **Spanish, German, or English** by default, and allows manual switching via a dropdown menu without reloading the page.
- **🌌 Parallax Starfield Background:** An HTML5 Canvas system renders a dynamically generated starfield that reacts to the user's scroll (parallax effect), adding depth and a "game-like" feel.
- **📱 Fully Responsive Design:** The layout automatically adapts to mobile devices, stacking elements cleanly and adjusting typography for readability on any screen size.
- **🪄 Glassmorphism Aesthetics:** UI cards and navigation menus use modern CSS backdrop-filters to create a sleek, frosted-glass effect over the dark theme.
- **🧭 Sticky Navigation:** A sticky menu allows for smooth scrolling between the different sections of the CV (Profile, Skills, Journey) from anywhere on the page.
- **✨ Micro-Animations:** Includes CSS hover effects, staggered fade-ins for skills, and a typing-text animation in the hero section.

## 🛠️ Technologies Used

This project was built entirely from scratch without the use of heavy frameworks or external libraries, prioritizing performance and load speed:

- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom styling, CSS variables, Flexbox, media queries, and keyframe animations.
- **Vanilla JavaScript (ES6):** Logic for the custom translation dictionary, intersection observers for scroll animations, and the canvas-based starfield.
- **SVG:** Scalable Vector Graphics for a crisp, transparent custom Favicon.
- **FontAwesome:** For lightweight, scalable iconography.
- **GitHub Pages:** For fast and reliable hosting.
